Developer Guides




OpenShift provides a modern container based hosting solution; allowing developers to focus on their application code. OpenShift is RedHat's cloud development Platform as a Service (PaaS). The free and open source cloud-based platform allows developers to create, test and run their applications and deploy them to the cloud - GICOE has deployed an image of OpenShift on Lilly's internal network, providing you with direct access to Lilly's vast IT ecosystem
